# LibertMafia

마피아 미니게임용 Paper 플러그인입니다.

## 주요 설정

- `debug.enabled`: 디버그 모드 여부 (`/디버그`로 토글됨)
- `police.station.*`: 경찰서 블럭 위치

## 빠른 사용

- 서버에서 `/시작`으로 게임을 시작합니다.
- 크리에이티브 모드에서 이름표로 블럭을 우클릭하면 경찰서 위치가 지정됩니다.

## 단서 시스템

- 크리에이티브 모드 + 부싯돌 좌클릭: 첫번째 지점 지정
- 크리에이티브 모드 + 부싯돌 우클릭: 두번째 지점 지정
- `/시작` 최초 실행 시 범위 내 모든 상자를 비우고 랜덤 상자 2개에 "단서"를 배치
- 단서 3개를 제작대/인벤토리 제작 슬롯에 올리면 "암살 무기" 제작 가능
- 암살 무기는 플레이어를 우클릭해 1회 처치 가능
